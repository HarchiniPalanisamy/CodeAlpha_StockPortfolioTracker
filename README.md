# Stock Portfolio Tracker using Python

## Description

This is a simple Stock Portfolio Tracker developed using Python.

The program allows users to enter stock names and quantities, calculates the investment amount, and displays the total portfolio value.

It also saves the stock details and total investment value in a text file (`portfolio.txt`).

## Features

* User inputs stock names (Apple, Tesla, Google, Microsoft)
* User enters stock quantity
* Calculates investment amount
* Calculates total investment value
* Saves portfolio report in `portfolio.txt`
* Console-based application

## Technologies Used

* Python
* Dictionary
* Input/Output
* Loops
* If-Else Conditions
* File Handling
* Basic Arithmetic

## Available Stocks

* Apple
* Tesla
* Google
* Microsoft

## How to Run

1. Install Python
2. Open the project folder
3. Run `StockPortfolio.py`

## Output

The program generates:

* Total Investment Value
* `portfolio.txt` file containing stock details and calculations
